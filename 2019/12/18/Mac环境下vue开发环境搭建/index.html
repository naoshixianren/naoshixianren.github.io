Mac环境下vue开发环境搭建

发表于 2019-12-18

最近换了mac电脑，赶上公司其他项目需要协助搭建基于vue的前端开发框架，就在自己的电脑上折腾了一下，以此为记录

  1. 安装homebrew

    ​ homebrew是mac os平台下软件包管理工具，拥有安装、卸载、更新、查看、搜索等很多实用的功能。简单的一条指令，就可以实现包管理，而你不用关心各种依赖和文件路径的情况，十分的方便、快捷。

    • homebrew安装准备

      安装homebrew需要依赖Xcode命令行工具

    • homebrew安装

      ​ 网络上可以找到各种homebrew的安装命令，但笔者推荐，进入官网获取，homebrew官网地址：http://brew.sh/，首页的代码即为home安装命令

    进入到项目目录，输入安装命令后，设置各种配置信息

  1. 项目启动

    ​ 进入项目根目录，控制台执行npm run dev即可启动项目，启动完成后浏览器输入localhost:8080，能够正常显示vue页面，即安装成功
